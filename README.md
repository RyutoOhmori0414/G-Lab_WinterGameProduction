ルール<br/>
1.Assetを追加したい場合は要相談、無断追加はNG。
  OKが出た際はAssetStoreToolsに格納し、このReadmeに書き込む(書き方が分からない場合は大森に相談)<br/>
2.ゲームの更新があった場合は、プルリクエストを出して、大森まで報告すること<br/>
3.自分のブランチを分けて、自分のフォルダーにシーンを作ってから制作してください<br/>
